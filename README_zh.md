# AI Elements Vue Starter

[English](README.md) | 中文

本项目是一个为 Vibe Coding 打造的 Vue.js(Vue 3) 基础模板，旨在为开发者提供简单、开箱即用的起步环境。你可以基于此模板快速搭建、扩展自己的前端应用项目。
无论是学习、实验还是正式项目开发，AI Elements Vue Starter 都为你打下稳固、灵活的基础。

## 参考资料

- [Vue.js](https://cn.vuejs.org/): 易学易用，性能出色，适用场景丰富的 Web 前端框架。
- [Vite](https://cn.vite.dev/): 一款速度极快的前端构建工具，为下一代 Web 应用程序提供支持。
- [TypeScript](https://www.typescriptlang.org/zh/): 一种强类型编程语言，它建立在 JavaScript 之上，在任何规模上都为您提供更好的工具。
- [Vue Router](https://router.vuejs.org/zh/): 为 Vue.js 提供富有表现力、可配置的、方便的路由。
- [Tailwind CSS](https://tailwindcss.com/): 只需书写 HTML 代码，无需书写 CSS，即可快速构建美观的网站。
- [shadcn/ui](https://ui.shadcn.com/): 提供一套精心设计、可访问的组件与代码分发平台，兼容你喜欢的框架和 AI 模型。开源，开放代码。
- [shadcn-vue](https://shadcn-vue.com/): Vue 移植版本的 shadcn-ui。
- [AI Elements Vue](https://www.ai-elements-vue.com/): 基于 shadcn-vue 构建的组件库，助你更高效地开发 AI 原生应用。
